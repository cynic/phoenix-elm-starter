# Practical Elm Log

Code examples from following Practical Elm Book

Ensure you have Node.js installed 


Install elm: https://guide.elm-lang.org/install/elm.html

```
elm --version
0.19.1
```

Install `elm-format`: 
```
npm install -g elm-format
```

Ran the command:
```
elm init
```
See: https://elm-lang.org/0.19.1/init
